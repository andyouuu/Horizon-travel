# Горизонт — сайт туристического агентства

Учебный одностраничный сайт на HTML, CSS и JavaScript. Не требует установки Node.js или сборки.

## Как открыть сайт

### Способ 1 — двойной клик (самый простой)

1. Откройте папку `horizon-travel` в Finder.
2. Дважды щёлкните по файлу `index.html`.
3. Сайт откроется в браузере (Safari, Chrome и т.д.).

### Способ 2 — через Cursor

1. В Cursor: **File → Open Folder** и выберите папку `horizon-travel`.
2. Щёлкните правой кнопкой по `index.html` → **Reveal in Finder** → откройте файл в браузере.

### Способ 3 — локальный сервер (если картинки не грузятся)

В терминале:

```bash
cd ~/Projects/horizon-travel
python3 -m http.server 8080
```

Откройте в браузере: http://localhost:8080

## Структура проекта

| Файл / папка | Назначение                          |
|--------------|-------------------------------------|
| `index.html` | Разметка страницы                   |
| `styles.css` | Оформление и адаптивная вёрстка     |
| `script.js`  | Меню, прокрутка, форма заявки       |
| `images/`    | Фотографии (работают без интернета)|

Фото хранятся локально в папке `images/`, поэтому сайт корректно открывается двойным щелчком по `index.html`.

## Разделы сайта

- Главный экран (hero)
- Популярные направления
- Цены на туры
- Отзывы клиентов
- Форма заявки

Форма показывает сообщение об успехе локально — данные никуда не отправляются (это демо).
